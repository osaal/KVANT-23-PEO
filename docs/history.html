Päivityshistoria

Tällä sivulla näet verkkokirjan päivityshistorian. Yritän pitää tämän niin ajantasalla ja tarkkana kuin voin, mutta mahdolliset pienet korjaukset voivat vahingossa mennä sivu suun.

Julkaisunumerot ovat muodossa MAJOR.MINOR. MINOR-päivitykset voivat sisältää uusia lukuja, korjauksia ynnä muita viilauksia; MAJOR-päivitykset ovat suuria muutoksia kirjaan (joita ei taida tulla usein).

Jos päivitys koskee GitHubissa annettua Issueta, tai jos jonkun käyttäjän Pull Request on mergattu kirjaan1, päivitystiedoissa löytyy linkki kyseiseen Issueen tai Pull Requestiin.
